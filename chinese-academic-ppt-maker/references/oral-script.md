# Oral Script

## Speaker Notes Style

Speaker notes should sound like a real person speaking in a meeting:

- short sentences;
- clear transitions;
- no manuscript-style long clauses;
- define terms only when needed;
- mark what is verified and what is still uncertain.

## Slide Note Template

`这一页我主要想说明三点。第一，...。第二，...。第三，...。所以这页的结论是：...。`

## Sensitive Claims

When a claim could be challenged, phrase it with a boundary:

- `这里我先按当前复现结果说，不把它上升到临床应用。`
- `这部分目前更适合说是获益分层信号。`
- `因为测试集仍来自同一项 RCT，所以我不会称它为独立外部验证。`
- `这个变量定义还需要老师或数据字典确认。`

## Likely Questions Slide

For project meetings, include a hidden or backup slide:

| 老师可能问 | 简短回答 |
|---|---|
| 为什么不是外部验证？ | 测试中心未参与训练，但仍来自同一 RCT。 |
| 反事实怎么知道？ | 个体反事实不可观测，模型估计条件平均治疗效应。 |
| 是否能临床用？ | 目前不能，需要外部和前瞻性验证。 |

## Rehearsal Output

When asked to rehearse, provide:

- 1-minute version;
- 3-minute version;
- slide-by-slide cue cards;
- likely questions and answers;
- phrases to avoid.
